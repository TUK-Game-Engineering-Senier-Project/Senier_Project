# FBX 비주얼 스튜디오

> FBX SDK 다운로드 : https://aps.autodesk.com/developer/overview/fbx-sdk
:: VS2022가 붙은 최신 파일을 다운로드한다.

Visual Studio 2022 설정 참고 내용 : https://blog.naver.com/jung_cpp/221175217328

[1. 헤더 파일 관련 폴더 추가]
설치된 경로에 가면 "include" 폴더가 있으며 해당 폴더를 추가 포함 디렉터리에 추가 필요
1. 해당 프로젝트의 "속성" 으로 이동
2. "구성 속성 -> C / C++ -> 일반 -> 추가 포함 디렉터리" 항목을 선택 하여 "편집"
  - 해당 "include" 폴더의 전체 경로를 입력하여 추가

[2. 라이브러리(lib) 관련 폴더 추가]
FBX SDK 로 설치된 경로에 가면 "lib" 폴더가 있으며 해당 폴더를 추가 포함 디렉터리에 추가 필요
1. 해당 프로젝트의 "속성" 으로 이동
2. "구성 속성 -> 링커 -> 일반 -> 추가 라이브러리 디렉터리" 항목을 선택 하여 "편집"
  - lib 폴더는 x86 / x64 폴더와 debug / release 폴더 등으로 구분되어 있음
  - 자신의 Visual Studio 설정 에 따라 x86 / x64 와 debug / release 경로를 확인하여 입력
 
[3. lib 파일 설정]
※주의 사항 : "C / C++ -> 코드 생성 -> 런타임 라이브러리" 의 설정과 추가 종속성 에 설정한 lib 파일 종류가 맞지 않는 경우 오류가 발생함
1. "C / C++ -> 코드 생성 -> 런타임 라이브러리" 의 설정을 "다수 스레드 디버그(/MTd)" 로 설정
2. "링커 ->  입력 -> 추가 종속성" 에
libfbxsdk-mt.lib
libxml2-mt.lib
zlib-mt.lib
추가
